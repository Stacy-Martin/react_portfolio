## Installation and Useage

To use this app, first run `npm install` and then install the following depenencies:

~~~
"jquery": "^3.6.0",
"popper.js": "^1.16.1",
"react": "^17.0.2",
"react-bootstrap": "^1.6.1",
"react-dom": "^17.0.2",
~~~

Initialize your session by running `npm start`.
